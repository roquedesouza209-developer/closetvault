# ClosetVault

Built to explore how secure and efficient file storage systems can be designed for everyday use.

ClosetVault is a cloud storage and synchronization web application that allows users to securely store, manage, upload, download, and share files across devices.

## Features
- File upload and download
- Secure storage system
- Cross-device synchronization
- Offline access (planned)
- File sharing

## Status
Concept and structure created. Further development planned.
